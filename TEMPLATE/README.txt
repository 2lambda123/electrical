README:
I have not found a convenient way to save and load template field names for components. 
for now do the following: open kiCAD -> preferences -> Template Field Names. Then add:

Manufacturer
MFG. P/N
Vendor
Vendor P/N
Tolerance (%)

All default values should be blank, except DigiKey. 
All should be invisible except Tolerance

*Change your default text size to 41 mil

****
To load the template into the test project:
click on the Page settings via GUI or file->page settings. Click 
"browse" at the bottom and find the .wks file.
